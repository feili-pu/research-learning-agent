# Changelog

## 0.16.0 - 2026-06-07

- Added Windows one-click local startup scripts: `scripts/start.ps1`, `scripts/stop.ps1`, and `.cmd` wrappers.
- Added `scripts/check-env.ps1` for conda, Node, npm, `.env`, LLM, dependency, and port checks.
- Added a GitHub Actions CI template for backend pytest and frontend build.
- Added LLM candidate reranking, evidence coverage guardrails, library duplicate merge, batch delete API, and BibTeX/RIS/CSV export.

## 0.15.0 - 2026-06-07

- Added external literature discovery and metadata import.
- Added LLM query parsing, retrieval tracing, paper-first retrieval, topic gates, and evaluation metrics.
